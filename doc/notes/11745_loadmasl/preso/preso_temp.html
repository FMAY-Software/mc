18/24

package references and naked OOA of MASL

  • A package reference is a package linked to another package in another location (package or project).

    • Package references allow reuse and import of subsystems.

  • The term naked OOA refers to a class model with no state or functional activity intended to be used as the schema for multiple projects.

    • A meta-model is most useful as a naked class model.
